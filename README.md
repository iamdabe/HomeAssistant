# HomeAssistant
Scripts, automations, blueprints for my home assistant setup

## Blueprints

**AWTRIX Simple Pollen** [awtrix_simple_pollen.yaml](https://github.com/iamdabe/HomeAssistant/blob/main/blueprints/awtrix_simple_pollen.yaml)

This blueprint shows the current pollen levels on an AWTRIX device. Uses different icons to depict current pollen levels. Uses 4 separate sensors for the current pollen levels which works with the default HA Tomorrow io integration. This blueprint is modified from [jeeftor](https://github.com/jeeftor/HomeAssistant) excellent awtrix pollen template available here [awtrix_pollen_template.yaml](https://github.com/jeeftor/HomeAssistant/blob/master/blueprints/automation/awtrix_pollen_template.yaml)
